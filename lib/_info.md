# Namespaces & keys
All are strings (namespace, key, nskey, partkey)

## Namespace
*'domain', 'error', 'domain/biography', ...*\
Hierarchies are possible

## Key

### Partial Key (partkey)
*'name', 'age', 'height', ...*\
NO specifications for these

### Namespaced Key (nskey)
*'domain/name', 'domain/age', ...*\
**$def** defines specs for these
